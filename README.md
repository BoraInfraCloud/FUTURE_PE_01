## 🧠 Prompt Framework: RCTF-X

All prompts follow the **RCTF-X Framework** — a 5-layer professional prompt engineering structure:

```
R — Role      → Assigns expert AI persona (senior healthcare copywriter)
C — Context   → Provides full business brief, audience, USPs, tone rules
T — Task      → Defines exact sections to generate with zero ambiguity
F — Format    → Sets word limits, section labels, variant requirements
X — eXamples  → Forbidden words list, formula requirements, quality benchmarks
```

## 💡 Key Learnings

1. **Layered prompts beat single prompts** — 5 specialised prompts produce far better output than one mega-prompt
2. **Role assignment is the highest-leverage technique** — assigning an expert persona raises output quality dramatically
3. **Format constraints force better copy** — "Max 8 words" produces sharper headlines than "write a headline"
4. **Healthcare copy = trust architecture** — forbidden words and tone rules are essential, not optional
5. **QA prompts catch mistakes humans miss** — a dedicated review prompt before deployment ensures professional-grade output
6. **Stitch AI + structured copy = fast results** — clear section labels allow AI builders to translate copy into working layouts instantly
